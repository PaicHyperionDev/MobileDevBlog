## Quick Start

### Create a new post

``` bash
$ hexo new "My New Post"
```

More info: [Writing](http://hexo.io/docs/writing.html)

### Run server

``` bash
$ hexo server
```

``` bash
$ hexo generate
```

### Deploy to remote sites

``` bash
$ hexo deploy
```
